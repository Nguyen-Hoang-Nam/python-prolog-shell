# Python Prolog Shell

Python Prolog shel is a Python CLI for runing prolog query in terminal.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Python Prolog shell.

```bash
  $ pip install python-prolog-shell
```

## Usage

```bash
  $ pyrolog
```
